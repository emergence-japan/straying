# Product Guide: Straying / Dice of Destiny

## 1. Vision & Purpose
「Straying / Dice of Destiny」は、ゲーム体験を通じてユーザーの内面的な変化（意識変容）を促し、その過程を精密に測定するためのプラットフォームです。柔軟なモジュール設計により、特定のゲーム部分を他のWebアプリケーションと容易に入れ替えられる構造を目指します。

## 2. Core Value Propositions
*   変容の測定: 事前・事後アンケートにより、体験を通じた心理的・意識的な変化を可視化します。
*   柔軟な統合能力: 既存のゲームエンジンに依存せず、外部のWebアプリを「ゲームパート」として取り込むことが可能です。
*   洗練されたUI: コンテンツに集中できるよう、背景装飾を排除したシンプルでクリーンなユーザーインターフェースを提供します。

## 3. Targeted Users
*   ワークショップ/研修参加者: 意識変容をテーマにした教育プログラムの受講者。
*   研究・調査対象者: 心理学的アプローチを用いた意識調査の協力者。

## 4. Key Functional Features
*   外部ゲーム連携インターフェース: 他のWebアプリをゲームパートとして埋め込み、連動させる機能。
*   多言語アンケートシステム: 日本語・英語に対応した事前/事後アンケート。
*   ミニマルなトップ画面: 背景画像を廃止し、操作性を最優先したランディングページ。
*   GAS連携: 回答データのGoogleスプレッドシートへの自動保存。

## 5. Success Criteria
*   他のWebアプリをゲームパートとして正常に統合し、アンケートフローを完結できること。
*   背景を排除したUIにより、ユーザーの離脱率が低下し、操作の迷いがなくなること。
